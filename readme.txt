[h1]每日惊喜 Daily Surprise[/h1]

[h2]模组介绍[/h2]
"每日惊喜"是一个为《饥荒联机版》制作的趣味模组。每天开始时，系统会随机选择指定数量的玩家获得一个特殊效果，为游戏增添变化和乐趣。

[h2]主要特点[/h2]
[list]
[*] 每天随机选择玩家获得特殊效果
[*] 可配置获得BUFF的玩家数量（1-5人或所有人）
[*] 10种有趣的BUFF效果
[*] 可配置BUFF持续时间（半天、1天或2天）
[*] 系统通知谁获得了每日惊喜
[/list]

[h2]BUFF效果列表[/h2]
[list]
[*] 超级速度：移动速度加倍
[*] 巨人化：体型变大
[*] 饥饿加速：饥饿速度加倍
[*] 幸运日：击杀生物有几率获得额外掉落物
[*] 夜视能力：夜间视野提升
[*] 饥饿减缓：饥饿速度减半
[*] 随机传送：有几率随机瞬移
[*] 生物朋友：生物不主动攻击
[*] 小矮人：体型变小
[*] 彩虹光环：发出彩色光芒
[/list]

[h2]配置选项[/h2]
[list]
[*] BUFF持续时间：可选择半天、1天或2天
[*] 随机选择玩家数量：可选择1-5人或所有人
[/list]

[h2]使用说明[/h2]
每天开始时，系统会随机选择指定数量的玩家获得BUFF。被选中的玩家会收到提示，告知获得了什么效果。所有玩家都会收到系统通知，告知谁获得了每日惊喜。BUFF效果会在设定时间后自动消失。

[h2]注意事项[/h2]
[list]
[*] 每个玩家每天只能获得一个随机BUFF
[*] 重新进入游戏会保持当天的BUFF效果
[*] 服务器和客户端都需要安装此模组
[*] 仅支持饥荒联机版，不支持单机版
[/list]
